    Para tal escreva no arquivo index.html a tag necessário para criar um formulário, declarando 2 de seus atributos essenciais:

  • O atributo method que tenha POST como valor
  • O atributo action que tenha resultado.html como valor

    • Resolução: form action="resultado.html" method="POST"

    Form 3 - Campo de Texto

    Queremos um campo dentro do formulário para que o usuário possa inserir seu nome. Para conseguirmos isso é necessário inserir duas tags dentro do formulário:

  • A tag "label", para informar ao usuário que deve inserir seu nome. A mesma deverá ter o atributo for com o valor nome.
  • A tag "input" com atributo type com o valor text (para que o usuário possa inserir seu nome) e o atributo id com o valor nome. Por último, declare o atributo name com o valor nome já que se este atributo não existir, não se enviará o campo como resultado.

    • Lembre-se que se estiverem bem configurados os atributos for da tag "label" e o atributo id da tag "input", ao clicar na "label" o cursor para escrever se posicionará automaticamente no "input" relacionado.

    Form 2 - Tem Pet?

    Vamos criar um formulário para perguntar se o usuário possui ou não um bicho de estimação. Para tanto, dentro do formulário que foi fornecido:

  • crie um "label" com o texto "Você tem um pet?"
  • adicione dois radio-button para dar a opção "Sim" e a opção "Não"
  • configure o atributo name="pet" nos dois "input" para que entenda que estamos falando do mesmo campo.
  • Para poder verificar logo qual foi a opção escolhida pelo usuário não se esqueça de adicionar o atributo value em cada "input". Para a opção "Sim" o valor deverá ser la letra s, e para a opção "Não" a letra n.

    Form 2 - selecionando mês

    Vamos criar um pequeno formulário com um drop-down (menu suspenso) para que o usuário possa selecionar o mês de seu aniversário.

  • Para isso, precisaremos criar a tag "select" com o atributo e valor name="mes". Dentro do "select" precisará criar 12 tags "option", uma para cada mês, declarando em cada caso o texto a ser lido pelo usuário: Janeiro, Fevereiro, Março, etc...
  • Não se esqueça, também, de escrever o atributo value em cada tag "option". Mas atenção, a intenção é que, como valor, esse atributo receba o número que representa o mês no calendário. Exemplo: 1 se refere à Janeiro, 2 se refere à Fevereiro, e assim por diante..
